Privacy Policy

🤖 Bot: ZR Partner
📅 Effective Date: July 8, 2026
This Privacy Policy explains what data the bot processes on Discord servers, why it is processed, and how long it is retained.

📋 Data We Collect

  • 👤 User IDs (server owners/representatives, realizators, partners)
  • 🏠 Server (Guild) IDs, including IDs of partner servers referenced in invite links
  • 💬 Channel IDs used for the partnership system and the log channel
  • 🎭 Role IDs (realizator role, partner role, leave/"urlop" role, permission checks)
  • 🔗 Discord invite links submitted in the designated partnership channel
  • 🤝 Partnership records: which realizator handled which partnership, the partner's user ID, the associated invite link, the date, and the reward rate assigned to that partnership
  • ⏱️ Norm/quota tracking data for realizators: cycle start time, whether a reminder was already sent, and the count of partnerships completed within the current cycle
  • 🚫 A local mirror of a blacklist (blocked server IDs, server names, and block reasons), synced from a separate, independently operated blacklist bot
  • 🎁 Completed-partnership counters used to determine when a realizator has reached the threshold for a reward payout
The bot does not use the Presence Intent (online/offline status of other users) — this is not required by, and is not used by, any bot feature.

Message Content — What Exactly Is Analyzed

The bot requires the Message Content Intent because the partnership system needs to detect Discord invite links posted in the designated channel. The bot only extracts the characteristics listed below — it does not permanently store the full text of messages:

  • 🔗 Presence and value of a Discord invite link (discord.gg / discord.com/invite) — used to register or verify a partnership
  • 🙋 Any user mentioned in the message — used to identify the partner associated with the invite
The invite link itself is retained as part of the partnership record (see above) so it can be shown to server staff and, if a partnership ends, removed. It is not otherwise stored or reproduced elsewhere.

🗄️ Data Storage

Data is stored locally on the bot's hosting server (SQLite databases) and used exclusively to support the features described in this Policy. Partnership records are automatically removed once a partnership ends (e.g. when a partner leaves the server). The blacklist mirror is fully replaced every time the source blacklist bot broadcasts an update, so it never accumulates stale entries.

🤝 Data Sharing

  • ✗ Data is never sold or shared with third parties
  • ✗ Data is never used for advertising purposes outside the server
  • ✗ ZR Partner does not automatically communicate or exchange data with the ZR Points bot — reward points are credited manually by server staff, based on the completed-partnership count shown by this bot, after the realizator opens a payout ticket
  • ✓ Blocked-server data is received from a separate, independently operated blacklist bot, solely to enforce partnership restrictions
  • ✓ Data may be visible to server administrators as part of standard moderation features (e.g. the partnership log channel)

👶 Children

The bot is intended to be used in accordance with Discord's Terms of Service, which require users to be at least 13 years old (or the minimum age required in their country). We do not knowingly collect data from users below this age.

✋ Your Rights

You may request deletion of your data (partnership records, norm/cycle history) at any time by contacting server staff through the ticket system. You may also stop using the bot at any time — your data will no longer be actively collected.

✏️ Changes to This Policy

We may update this Privacy Policy from time to time. Significant changes will be announced on the server. Continued use of the bot after changes take effect constitutes acceptance of those changes.

✉️ Contact

For any questions regarding privacy or data handling, please contact server staff through the ticket system.
