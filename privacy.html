🔒 Privacy Policy

🤖 Bot: Zimowy Partner

📅 Effective Date: July 8, 2026

This Privacy Policy explains what data Zimowy Partner collects from Discord servers, why it's collected, and how it's used to run the server-partnership program.

📋 Data We Collect

  • 👤 User IDs — of members who take part in or manage partnerships
  • 🏠 Server (Guild) IDs — of servers using the partnership system
  • 💬 Channel IDs — the designated partnership and log channels
  • ✉️ Message IDs — of partnership posts, so they can be updated or removed automatically
  • 🎭 Role IDs — used to grant or remove partnership-related roles automatically
  • 🔗 Invite links — posted in the partnership channel, used to identify the partner server
  • 📊 Partnership statistics — number of partnerships, timestamps, and assigned rates
  • 🚫 Blacklist entries — server ID, server name, and reason, mirrored from a connected moderation bot

We do not read or store the general content of your messages. Message content is only inspected inside the designated partnership channel, solely to detect invite links and mentioned users — nothing outside that channel is processed.

⚙️ How We Use This Data

  • 🤝 Matching and recording partnerships between servers
  • 🏅 Enforcing partnership limits and cooldowns
  • 📈 Tracking realizator quotas and automatically managing their roles
  • 💰 Calculating compensation rates tied to completed partnerships
  • 📬 Sending direct-message reminders and notifications
  • 🧾 Logging partnership activity to a private staff channel
  • ⛔ Synchronizing the server blacklist to block partnerships with flagged servers

🗄️ Data Storage

All data is stored locally in a database operated by the bot owner (not shared with any third-party data broker). Records are kept for as long as needed for the features above — for example, partnership records are automatically removed when a partnership ends or a member leaves the server.

🔗 Data Sharing

Data is never sold. It may only be shared with:

  • Discord, Inc. — as the platform the bot operates on
  • A connected blacklist bot, limited strictly to the sync fields listed above

👤 Your Rights

You can request access to, correction of, or deletion of your data at any time by contacting the bot operator below. Leaving a server or having the bot removed also stops any further collection for that server.

🔐 Security

Reasonable technical measures are used to protect stored data from unauthorized access. No online system can be guaranteed 100% secure.

📝 Changes to This Policy

This policy may be updated from time to time. Material changes will be announced through the support server listed below.
